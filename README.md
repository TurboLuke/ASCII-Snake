# ASCII-Snake

ASCII Snake written in C++

START:
```
| -  -  -  -  -  -  -  -  - |
|                           |
|                           |
|                           |
|    O        #             |
|                           |
|                           |
|                           |
|                           |
| -  -  -  -  -  -  -  -  - |
      ================
         ASCII-SNAKE
      ================
       USE ARROW KEYS
        TO PLAY/START
          THE GAME
      ================
```

GAME OVER:
```
| -  -  -  -  -  -  -  -  - |
|                         # |
|                      O  # |
|                #  #  #  # |
|                #          |
|                #          |
|                #          |
|                #          |
|                           |
| -  -  -  -  -  -  -  -  - |
      ================
         GAME  OVER
      ================
         SCORE: 90
        RESTART? y/n
```
